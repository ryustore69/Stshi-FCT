## 🚀 **Step-by-Step Installation**

### **Step 1: Download the Extension** 📥

<details>
<summary>📥 <strong>Click to expand download options</strong></summary>

#### **Option A: Direct Download (Recommended)**
1. Click the **"Code"** button on this repository
2. Select **"Download ZIP"**
3. Extract the ZIP file to a folder on your computer
4. Remember the folder location

#### **Option B: Git Clone (For Developers)**
```bash
git clone https://github.com/YOUR_USERNAME/satoshi-faucet-bot.git
cd satoshi-faucet-bot
```

#### **Option C: Chrome Web Store (Coming Soon)**
*The extension will be available on Chrome Web Store soon for easier installation*

</details>

### **Step 2: Install in Chrome** 🔧

<details>
<summary>🔧 <strong>Click to expand Chrome installation steps</strong></summary>

#### **Method 1: Using Address Bar**
1. Open **Google Chrome**
2. Type `chrome://extensions/` in the address bar
3. Press **Enter**

#### **Method 2: Using Menu**
1. Click the **three dots** menu (⋮) in Chrome
2. Go to **More tools** → **Extensions**

#### **Method 3: Using Extension Icon**
1. Right-click on any extension icon in the toolbar
2. Select **"Manage extensions"**

#### **Enable Developer Mode**
1. Toggle **"Developer mode"** switch (top right corner)
2. This enables advanced options for loading unpacked extensions

#### **Load the Extension**
1. Click **"Load unpacked"** button
2. Navigate to and select the extracted extension folder
3. Click **"Select Folder"**
4. ✅ Extension should now appear in the list!

</details>

### **Step 3: Pin Extension (Recommended)** 📌

<details>
<summary>📌 <strong>Click to expand pinning instructions</strong></summary>

1. Click the **puzzle piece icon** (🧩) in Chrome toolbar
2. Find **"Satoshi Faucet Bot"** in the list
3. Click the **pin icon** (📌) next to it
4. The extension icon will now be visible in your toolbar

</details>

### **Step 4: Setup & Start** 🎯

<details>
<summary>🎯 <strong>Click to expand setup instructions</strong></summary>

#### **Login to Satoshi Faucet**
1. Go to [https://satoshifaucet.io](https://satoshifaucet.io)
2. Login with your account credentials
3. Make sure you're on the main dashboard

#### **Configure Extension**
1. Click the **Satoshi Faucet Bot** icon in your toolbar
2. **Select Currency**: Choose your preferred cryptocurrency
3. **Enable Notifications**: Click ⚙️ → Enable desktop notifications
4. **Check Settings**: Verify all settings are to your preference

#### **Start Automation**
1. Click **"START BOT"** button
2. Bot will begin working automatically
3. You'll see real-time stats updating
4. 🎉 **You're now earning cryptocurrency automatically!**

</details>

## 🔧 **Troubleshooting**

### **❌ Common Issues & Solutions**

<details>
<summary>🚫 <strong>Extension Not Loading</strong></summary>

**Problem**: Extension doesn't appear after loading
**Symptoms**:
- Extension not visible in extensions list
- "Load unpacked" button doesn't work
- Error messages when loading

**Solutions**:
- ✅ **Check Chrome Version**: Ensure Chrome 88+ is installed
- ✅ **Enable Developer Mode**: Toggle must be ON
- ✅ **Refresh Extensions Page**: Press F5 or refresh
- ✅ **Restart Chrome**: Close and reopen Chrome completely
- ✅ **Check Folder Contents**: Ensure all files are present
- ✅ **Try Different Folder**: Move extension to different location

**Still Not Working?**
- Check Chrome console for error messages (F12)
- Try loading in incognito mode
- Disable other extensions temporarily

</details>

<details>
<summary>🤖 <strong>Bot Not Working</strong></summary>

**Problem**: Bot starts but doesn't perform actions
**Symptoms**:
- Bot shows "running" but no actions
- Stats not updating
- No notifications appearing

**Solutions**:
- ✅ **Login First**: Must be logged into Satoshi Faucet
- ✅ **Correct URL**: Ensure you're on satoshifaucet.io
- ✅ **Refresh Page**: Press F5 and restart bot
- ✅ **Check Console**: Press F12 → Console for errors
- ✅ **Verify Permissions**: Check extension permissions
- ✅ **Restart Extension**: Disable and re-enable extension

**Debug Steps**:
1. Open browser console (F12)
2. Look for error messages
3. Check if extension is communicating with page
4. Verify all required elements are present

</details>

<details>
<summary>📊 <strong>Stats Not Updating</strong></summary>

**Problem**: Statistics don't update or reset unexpectedly
**Symptoms**:
- Stats stuck at 0
- Stats reset to 0
- Inconsistent stat display

**Solutions**:
- ✅ **Stats Auto-save**: Data is saved in browser storage
- ✅ **Use Refresh Button**: Click refresh in popup
- ✅ **Check Extension Status**: Ensure extension is running
- ✅ **Restart Extension**: Disable and re-enable
- ✅ **Clear Browser Cache**: Clear Chrome cache and cookies
- ✅ **Check Storage**: Verify browser storage isn't full

**Data Recovery**:
- Stats are stored locally in Chrome
- Clearing browser data will reset stats
- Export logs before clearing data

</details>

<details>
<summary>🔔 <strong>Notifications Not Showing</strong></summary>

**Problem**: Desktop notifications don't appear
**Symptoms**:
- No notification popups
- Notification setting enabled but not working
- Chrome notification permission denied

**Solutions**:
- ✅ **Check Chrome Permissions**: Go to Chrome Settings → Privacy → Site Settings → Notifications
- ✅ **Enable for Extension**: Allow notifications for the extension
- ✅ **Check Extension Setting**: Verify notification is enabled in extension
- ✅ **Test Notifications**: Use Chrome's notification test
- ✅ **Restart Chrome**: Close and reopen Chrome
- ✅ **Check System Settings**: Ensure system notifications are enabled

**Permission Setup**:
1. Go to `chrome://settings/content/notifications`
2. Add `chrome-extension://[extension-id]` to allowed sites
3. Or set to "Ask before sending" and allow when prompted

</details>

<details>
<summary>🌐 <strong>Page Not Loading</strong></summary>

**Problem**: Satoshi Faucet page doesn't load or has issues
**Symptoms**:
- Page won't load
- Captcha not appearing
- Elements not found

**Solutions**:
- ✅ **Check Internet**: Ensure stable internet connection
- ✅ **Clear Cache**: Clear browser cache and cookies
- ✅ **Disable Ad Blockers**: Temporarily disable ad blockers
- ✅ **Try Incognito**: Test in incognito mode
- ✅ **Update Chrome**: Ensure Chrome is up to date
- ✅ **Check Site Status**: Verify Satoshi Faucet is working

</details>
